Imports: `json`, `os`, `datetime` + `timezone` from `datetime`, `Dict`/`Any` from `typing`, plus relative imports `from .storage import ensure_data_dir` and `from .config import DATA_DIR`. Define ONE public function `aggregate_current_month() -> Dict[str, Any]`:

1. Call `ensure_data_dir()` (idempotent).
2. Compute `month_start_iso` = first-day-of-current-month ISO string (UTC), via `datetime.now(timezone.utc).replace(day=1, hour=0, minute=0, second=0, microsecond=0).isoformat()`.
3. Initialize accumulators: `total = 0.0`, `upstream_total = 0.0`, `queries = 0`, `by_profile: Dict[str, Dict[str, Any]] = {}`.
4. For each `*.json` filename in `os.listdir(DATA_DIR)`:
   - Open + `json.load` (synchronous — accepted per Pitfall 6 lock).
   - Skip files whose `data.get('created_at', '')` < `month_start_iso`.
   - Iterate `data.get('messages', [])`; skip non-assistant messages.
   - Extract `cost = (msg.get('metadata') or {}).get('cost') or {}`.
   - Read `t = float(cost.get('total') or 0.0)`, `u = float(cost.get('upstream_total') or 0.0)`. If both are 0.0, skip (legacy message or zero-cost run).
   - Increment `total += t`, `upstream_total += u`, `queries += 1`.
   - Read `profile = (msg.get('metadata') or {}).get('profile', 'unknown')` and accumulate into `by_profile[profile]` bucket with sub-keys `total`, `upstream_total`, `queries` (use `setdefault` per the PATTERNS.md template).
5. Return a dict with three top-level keys: `current_month` (sub-keys `total_usd` rounded to 4 decimals, `upstream_total_usd` rounded to 4, `queries` int, `by_profile` dict), `current_session_estimate_for_cap` (sub-keys `remaining_pct` = `max(0.0, 1.0 - total/100.0)` and `cap_usd` = `100.0`), and `currency` = literal string `"USD"`.

Defensive `.get() or {}` and `float(x or 0.0)` chains throughout — malformed legacy JSON files MUST NOT crash the aggregation. Per CLAUDE.md: 4-space, snake_case, no docstrings unless WHY non-obvious; a single one-line module docstring at the top of `stats.py` is acceptable ("Cost stats aggregation — read-only walk over data/conversations/*.json").

No `response_model` (plain-dict return, FastAPI serializes). No try/except (matches the existing route style at 83-86 — unhandled errors surface as HTTP 500). Add `from . import stats` to the existing relative-import block alongside the existing `from . import storage`.
