**`useTheme` hook contract** (`frontend/src/hooks/useTheme.js`):

```typescript
export function useTheme(): {
  theme: 'light' | 'dark';            // current state — reactive
  setTheme: (next: 'light' | 'dark') => void;  // explicit set + persist + stop following system
  toggle: () => void;                  // light <-> dark + persist
}
```

Behavioural contract:
- `readInitialTheme()` mirrors the index.html FOUC blocker — reads `localStorage.theme` first, falls back to `matchMedia('(prefers-color-scheme: dark)')`.
- `useEffect` applies `document.documentElement.setAttribute('data-theme', theme)` on every theme change.
- A second `useEffect` subscribes to `matchMedia change` only while `followSystem === true` (i.e., user has not made a manual choice). Calling `setTheme` flips `followSystem → false`.
- `setTheme` writes `localStorage.setItem('theme', next)` inside try/catch (private browsing tolerance).
